# Senza Giri Travel

Web app gratuita per raccogliere dati di viaggio, tappe desiderate e fonti online, e trasformarli in un prompt ottimizzato da usare con ChatGPT.

## Obiettivo

L'utente inserisce:

- destinazione e date;
- hotel/base;
- orari di arrivo e partenza;
- tappe desiderate;
- prenotazioni fisse;
- appunti o testi copiati da TikTok, Pinterest, blog e articoli.

La web app genera un prompt strutturato per ottenere un itinerario sequenziale, realistico e senza giri inutili.

## Versione attuale

MVP 0.1 - Travel Prompt Builder.

Funziona senza backend, senza login e senza API a pagamento.
